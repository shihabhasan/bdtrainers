== Frequently Asked Questions ==

= How to setup Theme? =
1. Upload the `Invert-lite` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Invert Options to change theme options.

= How to set Theme Front Page? =
1. The Front Page will get set automaticaly after theme activation.

= How to set Blog Page? =
1. Create a page for Front Page and another for Blog Page.
2. See Settings -> Reading -> Choose the created Page as Front page and other Page for Posts page from drop-down.
Note : You can also set this setting from Appearance -> Cusomize -> Static Front Page.

= Where is Navigation Menu? How to set a dropdown menu? =
To set up the navigation menu go to "Appearance >> Menus" create or use existing menu as Main Navigation.
To create dropdown menu drag the menu items somewhat right and they will automatically get set as dropdown.

= How to create a full width page? =
1. Go to Pages.
2. Click Add New or Edit an existing page.
3. Select "Full Width Template" from "Page Attributes >> Template".

= How to create a left sidebar page? =
1. Go to Pages.
2. Click Add New or Edit an existing page.
3. Select "Left Sidebar Template" from "Page Attributes >> Template".
